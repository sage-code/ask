Ask Project

This is a quiz web app. It enable members to take quizzes that are pre-loaded in a database. The purpose of this project is to learn how to build a dynamic web app. We will use the app for making an exam, it will not replace Google prep quizzes that we already have.

Documentation

For documentation we will use HTML5 and design diagrams to create a database structure. We will initialize the database using JSON data files. Data files will be generated using AI. There are no input dialogs except the Quiz dialog.

Tech stack

Following tech stack and tools can be used to build a wide variety of web applications, from simple static websites to complex dynamic applications. The specific tools and technologies you need will depend on the specific requirements of your project.

  • HTML + CSS

    • HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) are the core technologies used to create web pages. HTML is used to define the structure of a web page, such as headings, paragraphs, images, and links. CSS is used to style the appearance of a web page, such as the font, color, and layout.

  • JavaScript

    • JavaScript is a programming language that is used to add interactivity to web pages. For example, JavaScript can be used to create animations, validate forms, and handle user input.

  • MariaDB

    • MariaDB is a relational database management system (RDBMS) that is used to store and manage data for web applications. MariaDB is a fork of MySQL, and it is compatible with most MySQL applications.

  • Node.js

    • Node.js is a JavaScript runtime environment that allows developers to build back-end web servers and applications. Node.js is known for its scalability and performance.

  • Adonis

    • Adonis is a full-stack web framework for Node.js that provides a number of features to help developers build robust and scalable web applications. Adonis is based on the Laravel PHP framework.

  • Svelte

    • Svelte is a modern JavaScript framework that is known for its performance and ease of use. Svelte is a compiler-based framework, which means that it converts Svelte code into vanilla JavaScript code at compile time. This makes Svelte applications very fast and lightweight.

  • GitHub

    • GitHub is a code hosting platform that can be used to collaborate on software development projects. GitHub provides features such as version control, issue tracking, and pull requests.

  • VS Studio

    • Visual Studio is a popular integrated development environment (IDE) that can be used to write, debug, and test code. Visual Studio supports a variety of programming languages, including C++, Java, Python, and JavaScript.
