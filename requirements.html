Requirements

This Quiz App is designed to evaluate the knowledge and understanding of users who have completed the Software Engineering and Programming Fundamentals and/or the Programming Languages courses offered by our free code camp.

Functional Requirements

Registration/Login

  • Users should be able to create a new account with a valid email address and password.
  • Users should be able to log in to their existing account.

Quiz Taking

  • Users should be able to select the course they wish to take the quiz for.
  • The quiz should feature a mixture of multiple-choice, true/false, and short-answer questions.
  • Questions should be randomly generated from a question bank to provide different quiz experiences for each user.
  • After completing the quiz, users should receive a score and a report detailing which questions they answered correctly or incorrectly.

Non-Functional Requirements

Interface Design

  • The user interface should be user-friendly and responsive.
  • The interface should showcase Free Code Camp's branding and design elements.

Security Requirements

  • User passwords should be securely stored using a hashing algorithm.
  • User sessions should be authenticated and encrypted.
  • The quiz system should be protected against SQL injections and other possible security threats.

Performance Requirements

  • The quiz system should be fast and responsive.
  • The system should be able to handle multiple concurrent users.
  • The quiz system should be able to handle a large number of questions and answers stored in the database.

Conclusion

The Free Code Camp Quiz App should be designed with both functional and non-functional requirements in mind to provide a secure, user-friendly, and responsive quiz experience for users.
