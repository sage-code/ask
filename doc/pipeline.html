Software Pileine

A developer pipeline is a set of automated processes that help developers build, test, and deploy software applications in a more efficient and reliable manner. It include server side and client side components required by software architecture and platform. We chose Linux platform.

What is a Pipeline>

A typical software development pipeline involves the following stages.

  • Source code management: This stage involves managing the source code by using version control systems such as Git, SVN, etc.
  • Build: This stage involves compiling the source code into an executable format that can be run by the user.
  • Testing: This stage involves running automated tests on the software application to ensure it meets the functional and non-functional requirements.
  • Deployment: This stage involves releasing the software application into a production environment for the end-users to use.
  • Monitoring: This stage involves monitoring the software application in production to ensure it is working as expected and to detect any issues or bugs.

Dev Environment

To install all the necessary dependencies to build a quiz app with Svelte, Vite, Jest, NodeJS, Adonis, and MariaDB, along with Visual Studio Code, on Ubuntu, follow these steps:

Install Node.js:

First, we need to install Node.js and npm on your system. To do this, follow these steps:

  1. Open a terminal window by pressing "Ctrl+Alt+T" or searching for "Terminal" in the Activities menu.
  2. Update the package list by running:
sudo apt update
  1. Install Node.js using the following command:
sudo apt install nodejs
  1. Verify that Node.js and npm are installed by running the following commands:
node -v npm -v

Install VS Code:

Next, we need to install Visual Studio Code to develop our project. Follow these steps to install VS Code:

  1. Download the .deb file from the official website: https://code.visualstudio.com/download
  2. Open a terminal window and navigate to the directory where you downloaded the file:
cd Downloads
  1. Install the .deb package using the following command:
sudo apt install ./.deb

Install MariaDB:

MariaDB is an open-source database management system that we will be using for our project. Follow these steps to install MariaDB:

  1. Install the MariaDB server and client by running the following command:
sudo apt install mariadb-server mariadb-client
  1. Secure your installation by running:
sudo mysql_secure_installation

Install Svelte, Vite and Jest:

To install Svelte, Vite, and Jest, run the following commands in your terminal:

  1. Install Svelte globally:
sudo npm install -g svelte
  1. Install Vite globally:
sudo npm install -g vite
  1. Install Jest globally:
sudo npm install -g jest

Install Adonis CLI:

AdonisJS is a Node.js web framework. To install the Adonis CLI, run the following command in your terminal:

sudo npm i -g @adonisjs/cli

Once you've completed the above steps, you're all set up to develop a quiz app with Svelte, Vite, Jest, NodeJS, Adonis, and MariaDB, along with Visual Studio Code.
