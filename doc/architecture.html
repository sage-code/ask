Ask Cloudflare Architecture

This app is built as a Cloudflare-ready quiz application using Svelte for the front-end, quiz JSON files for content, and a Cloudflare Worker API for registration and result storage.

Project architecture

  • Frontend: Svelte single-page app.
  • Quiz content: JSON files in public/quizzes/.
  • Backend: Cloudflare Worker in worker/index.js.
  • Database: Cloudflare D1 stores users and results.
  • Deployment: static front-end build to dist/ and Worker deployment with Wrangler.

How it works

  1. User registers as a Mentor or Student.
  2. The app loads quiz questions from a JSON file by slug.
  3. User submits the quiz results.
  4. The Worker saves the results in the D1 database.
  5. Recent results are returned and shown in the UI.

Cloudflare setup

  1. Create a D1 database in the Cloudflare dashboard.
  2. Add the D1 binding to wrangler.toml.
  3. Run the schema in scripts/schema.sql.
  4. Deploy with wrangler publish.

Shared quiz URL

Each quiz is accessible with a URL like:

https://your-site.pages.dev?quiz=programming-basics

Important files

  • src/App.svelte - quiz UI and API integration.
  • public/quizzes/programming-basics.json - sample quiz data.
  • worker/index.js - Cloudflare Worker endpoints.
  • scripts/schema.sql - database schema for D1.
  • wrangler.toml - Cloudflare worker configuration.

Deployment

npm install
npm run build
wrangler publish
